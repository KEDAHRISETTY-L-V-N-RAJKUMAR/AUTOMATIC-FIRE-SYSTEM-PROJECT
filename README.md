## 📌 Overview
This project presents an **IoT-based automatic fire detection and extinguishing system integrated with a robotic car** using ESP32.

The system detects fire conditions using sensors and **autonomously moves a robotic vehicle towards the fire source** to perform extinguishing action, making it a smart and real-time safety solution.

---

## 🎯 Objectives
- Detect fire and hazardous gas conditions  
- Enable automatic response using actuators  
- Integrate mobility using a robotic platform  
- Ensure real-time monitoring and safety  

---

## 🧰 Components Used
- ESP32 Microcontroller  
- MQ-2 Gas Sensor  
- Flame Sensor  
- DHT11 Temperature Sensor  
- Motor Driver (for robotic car)  
- DC Motors (Robot Movement)  
- Servo Motor / Water Pump  
- Buzzer  

---

## ⚙️ Working Principle
- Sensors continuously monitor environmental conditions  
- If fire/gas is detected:
  - Buzzer is activated  
  - Robotic car moves toward the fire source  
  - Water pump/servo extinguishes the fire  
- System operates **autonomously using ESP32 control**
